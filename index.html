Resumo

Sistema de Reserva Online – Mercadinho Pereira. 2025. 35f. TCC (Técnico em Informática) – Colégio SR Millennium.
O comércio eletrônico moderniza pequenos negócios. Este trabalho descreve análise, projeto e desenvolvimento de um sistema de reserva online para o Mercadinho Pereira, permitindo que clientes reservem produtos pela internet e retirem na loja, sem pagamento online. Utilizamos modelagem UML, front-end responsivo (HTML, CSS, JS), back-end em PHP e banco MySQL. Resultado: site com catálogo, reservas, e-mail de confirmação e área administrativa.

HTML5 CSS3 JavaScript PHP 8 MySQL UML

Conclusão

O TCC atingiu o objetivo de criar um sistema de reserva online para o Mercadinho Pereira. Utilizamos modelagem UML, HTML, CSS, JS, PHP e MySQL. A solução mostra como pequenos negócios podem usar tecnologia acessível para melhorar atendimento, controle de estoque e presença digital. O sistema funciona conforme planejado e representa um avanço na modernização do comércio local.
